details about the workshop

Hunting vulnerabilities in Node apps

Practical workshop where we are going to use techniques and tools to discover vulnerabilities in a 100% Javascript application (Angular + Node) based on OWASP JuiceShop Project. Once vulnerabilities are identified we are going to "patch" the app and as conclusion we are going to review security best practices for Node.js apps.
